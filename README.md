# What's this?
This contains Java source code for several plugins I wrote for my Minecraft server running on PaperMC. The source is provided as is and requires the Paper/Bukkit API to build. Last time I updated these was for 1.20

# What's in it?
This repo contains several plugins:
* Twoolio : Adds a tool/weapon XP and upgrade system. Makes all tools unbreakable by default. Breaking blocks/attacking mobs gains XP. When a new level is gained, the player may right click on the tool to open an upgrade menu to apply an enchantment or upgrade the tier of the tool.
* WarpsGui-- : A decompiled plugin largely reworked. Removed features I didn't need and added BlueMap integration to add warp markers on the online map
* PocketShulkerBox+ : Allows opening of shulker boxes within the inventory by right clicking the box and not needing to place it in the world
* AnvilPenaltyForgiver+ : Resets the anvil cost for enchantments when the cost tips over the "Too expensive!" threshold.
* OPLevelShift : Allows the player to switch their Survival/Creative/Adventure/Spectator mode by pressing F3+F4 without needing to be OP on the server.
* TimeMaster : Allows using the pocket watch to set the time and weather with a simple menu
* XPMultiplier : A simple XP multiplier
